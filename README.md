# 宝宝成长记 — 0-12月龄婴幼儿成长管理小程序

## 项目简介

专为0-12月龄婴幼儿设计的成长管理小程序，通过"记录 → AI分析 → 个性化建议"的闭环，帮助宝妈科学育儿。核心差异化功能是AI驱动的宝宝天赋倾向评估与发展建议。

## 快速开始

### 1. 环境准备

- 微信开发者工具（最新版）
- 微信小程序账号（个人主体即可）

### 2. 配置步骤

1. 用微信开发者工具打开 `miniprogram/` 目录
2. 在 `project.config.json` 中替换 `appid` 为你的小程序 AppID
3. 准备 TabBar 图标（8个png文件）放入 `miniprogram/images/`

### 3. TabBar 图标规格

| 文件名 | 说明 | 尺寸 |
|--------|------|------|
| tab-home.png | 首页（未选中） | 81×81px |
| tab-home-active.png | 首页（选中） | 81×81px |
| tab-record.png | 记录（未选中） | 81×81px |
| tab-record-active.png | 记录（选中） | 81×81px |
| tab-assess.png | 评估（未选中） | 81×81px |
| tab-assess-active.png | 评估（选中） | 81×81px |
| tab-profile.png | 我的（未选中） | 81×81px |
| tab-profile-active.png | 我的（选中） | 81×81px |

图标建议使用粉色系（#FF6B81），未选中状态用灰色（#999999）。
