# 0-12月龄婴幼儿成长管理小程序 — 产品设计方案总览

## 项目概述

专为0-12月龄婴幼儿设计的成长管理小程序，以300份用户调研问卷为数据基石，通过"记录→AI分析→个性化建议"的闭环，帮助宝妈科学育儿。核心差异化功能是AI驱动的宝宝天赋倾向评估与发展建议。

## 核心数据流

```
用户记录数据 → 数据清洗编码 → 特征工程
    → AI评估模型 → 五维天赋画像
    → LLM报告生成 → 个性化建议
    → 用户执行建议 → 产生新数据
    → 模型迭代优化（闭环）
```

## V1.0 MVP 五大模块

1. **成长数据记录** — 身高/体重/头围 + WHO曲线
2. **发育里程碑对照** — 五大领域技能勾选 + 雷达图
3. **睡眠/喂养/便便记录** — 一键打卡 + 统计分析
4. **AI智能评估与天赋分析** — 五维画像 + 培养建议
5. **疫苗接种提醒** — 自动匹配 + 推送提醒

## AI天赋评估核心逻辑

基于多维输入数据（体格、发育里程碑、行为模式、互动记录），通过机器学习模型输出五维天赋画像：

| 天赋维度 | 评估依据 |
|----------|----------|
| 运动天赋 | 大运动里程碑达成速度、活动量 |
| 语言天赋 | 发音频次、语音反应、模仿能力 |
| 逻辑认知 | 因果理解、探索行为、注意力 |
| 艺术感知 | 对色彩/音乐/节奏的反应 |
| 社交情感 | 眼神交流、微笑频率、共情表现 |

## 关键约束

- MVP阶段功能 ≤ 5个模块
- 问卷回收周期 ≤ 7天
- 不强制收集位置信息
- 所有功能从问卷数据推导，不预设
- 覆盖新手宝妈和二胎宝妈两类用户
