# 0-12月龄婴幼儿成长管理小程序 — 数据清洗与需求分析框架

## 一、数据清洗规则

### 1.1 无效问卷剔除标准

| 规则编号 | 判定条件 | 处理方式 |
|----------|----------|----------|
| R1 | 答题总时长 < 60秒 | 剔除 |
| R2 | Q1/Q2/Q9/Q14任一关键字段为空 | 剔除 |
| R3 | 同一微信ID重复提交 | 保留最早提交，剔除后续 |
| R4 | Q14所有功能均打同一分数（如全选5或全选1） | 标记可疑，人工复核 |
| R5 | Q19/Q20回答为无意义字符（如"啊啊啊"、"111"） | 剔除 |
| R6 | IP地址/设备指纹异常（同设备 > 2次提交） | 剔除超限部分 |

### 1.2 预期清洗后样本分布

假设原始回收 350份，预计剔除 40-50份，保留有效样本 ≈ 300份。

有效样本需满足分层要求：
- 新手宝妈 ≥ 150份（50%）
- 二胎宝妈 ≥ 90份（30%）
- 其他 ≤ 60份（20%）

若某层不足，需在该渠道补投问卷。

## 二、需求权重计算方法

### 2.1 功能需求权重（基于Q14）

对每个功能项，计算加权平均分：

```
功能权重 = Σ(评分 × 选择人数) / 总有效样本数
```

示例：

| 功能 | 1分人数 | 2分人数 | 3分人数 | 4分人数 | 5分人数 | 加权均分 |
|------|---------|---------|---------|---------|---------|----------|
| 成长数据记录 | 5 | 10 | 40 | 100 | 145 | 4.23 |
| AI智能评估 | 15 | 25 | 60 | 100 | 100 | 3.88 |
| ... | ... | ... | ... | ... | ... | ... |

### 2.2 痛点权重（基于Q9排序）

```
痛点权重 = 第1名次数×3 + 第2名次数×2 + 第3名次数×1
```

| | 身高体重焦虑 | 睡眠焦虑 | 发育达标焦虑 | 喂养焦虑 | 早教焦虑 |
|---|---|---|---|---|---|
| 成长数据记录 | ★★★ | | ★★ | | |
| 发育里程碑对照 | ★★ | | ★★★ | | |
| 睡眠/喂养记录分析 | | ★★★ | | ★★★ | |

## 三、Kano模型分析

### 3.1 Kano问卷设计（附加于Q14之后的正反题对）

对每个功能，设置正向和反向两个问题：
- 正向："如果XX功能存在，您觉得？" → 喜欢/理所当然/无所谓/能忍受/不喜欢
- 反向："如果XX功能不存在，您觉得？" → 同上

### 3.2 Kano分类判定表

| 正向\反向 | 喜欢 | 理所当然 | 无所谓 | 能忍受 | 不喜欢 |
|-----------|------|----------|--------|--------|--------|
| 喜欢 | Q | A | A | A | O |
| 理所当然 | R | I | I | I | M |
| 无所谓 | R | I | I | I | M |
| 能忍受 | R | I | I | I | M |
| 不喜欢 | R | R | R | R | Q |

M=基本型, O=期望型, A=兴奋型, I=无差异, R=反向, Q=可疑

### 3.3 预期Kano分类结果（假设）

| 功能 | 预期分类 | 依据 |
|------|----------|------|
| 成长数据记录 | 基本型(M) | 宝妈普遍认为这是基础能力 |
| 发育里程碑对照 | 基本型(M) | 焦虑驱动，没有会不满 |
| 睡眠/喂养记录分析 | 期望型(O) | 有则满意，没有会失望 |
| AI智能评估 | 兴奋型(A) | 超预期的差异化功能 |
| 早教游戏推荐 | 兴奋型(A) | 锦上添花，惊喜感强 |
| 疫苗提醒 | 基本型(M) | 工具属性，必须有 |
| 宝妈社区 | 无差异(I) | 竞品已饱和，差异化低 |
| 在线问诊 | 期望型(O) | 有价值但非核心 |

## 四、功能优先级四象限图

以"用户选择频次"为X轴，"功能加权评分"为Y轴，绘制四象限图：

```
        高评分
          │
    Ⅱ 优化区  │  Ⅰ 核心区
    早教推荐  │  成长记录
    AI评估    │  发育对照
          │  睡眠喂养
  ──────────┼──────────  高频次
    Ⅲ 观望区  │  Ⅳ 效率区
    社区      │  疫苗提醒
    在线问诊  │  相册时间轴
          │
        低评分
```

**Ⅰ 核心区**（高频次 + 高评分）→ MVP必选
**Ⅱ 优化区**（低频次 + 高评分）→ MVP可选，重点优化体验
**Ⅲ 观望区**（低频次 + 低评分）→ 暂不开发
**Ⅳ 效率区**（高频次 + 低评分）→ 简单实现即可
