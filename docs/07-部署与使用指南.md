# 宝宝成长记 — 部署与使用指南

## 一、环境要求

| 工具 | 版本要求 | 说明 |
|------|----------|------|
| 微信开发者工具 | 最新稳定版 | 下载地址：developers.weixin.qq.com |
| 微信小程序账号 | 已注册 | 个人主体即可 |

## 二、部署步骤

### Step 1：导入项目

1. 打开微信开发者工具
2. 选择「导入项目」
3. 目录选择 `baby-growth-app/miniprogram/`
4. AppID 填入你的小程序 AppID（在 mp.weixin.qq.com 获取）

### Step 2：添加 TabBar 图标

在 `miniprogram/images/` 目录下放入 8 个图标文件：

```
images/
├── tab-home.png          # 首页（未选中，灰色）
├── tab-home-active.png   # 首页（选中，粉色 #FF6B81）
├── tab-record.png        # 记录（未选中）
├── tab-record-active.png # 记录（选中）
├── tab-assess.png        # 评估（未选中）
├── tab-assess-active.png # 评估（选中）
├── tab-profile.png       # 我的（未选中）
├── tab-profile-active.png# 我的（选中）
└── baby-default.png      # 默认宝宝头像
```

图标规格：81×81px，PNG 格式，透明背景。

推荐图标风格：线性图标（未选中）→ 填充图标（选中），颜色 #999999 → #FF6B81。

### Step 3：预览与调试

1. 点击开发者工具的「预览」按钮
2. 用微信扫码在手机上预览
3. 测试完整流程：填写宝宝信息 → 记录数据 → 查看里程碑 → 生成AI报告

## 三、数据存储说明

所有数据存储在用户手机本地（wx.setStorageSync），无需服务器或云开发环境。

## 四、测试数据建议

为了完整测试 AI 评估功能，建议录入以下测试数据：

| 类型 | 数量 | 示例 |
|------|------|------|
| 身高体重 | ≥3条 | 50cm/3.2kg, 58cm/5.1kg, 65cm/7.2kg |
| 睡眠记录 | ≥5条 | 每天2-3次，每次1-3小时 |
| 喂养记录 | ≥5条 | 母乳/奶粉，每天6-8次 |
| 便便记录 | ≥3条 | 黄色软糊状 |
| 里程碑勾选 | ≥10项 | 分布在不同维度 |

当记录数 ≥15 条时，AI 评估功能即可使用。

## 五、常见问题

**Q: TabBar 图标不显示？**
A: 检查 images 目录下图标文件名是否与 app.json 中一致。

**Q: 数据丢失？**
A: 数据存储在本地，卸载小程序或清除缓存会丢失。可在「我的」页面导出 CSV 备份。

**Q: AI 报告生成失败？**
A: 确认记录数据 ≥15 条。
